Instrucciones

  1. Crea la base de un sitio web usando una cuenta de Google, GitHub y opcionalmente Visual Sudio Code.

    Las instrucciones que se mencionan de git son:

    git config --global user.name tu_usuario_en_GitHub

    git config --global user.email tu_email_en_GitHub

    Para actualizar tu página cuando no han pasado 10 minutos del último despliegue, teclear

    Shift+Ctrl+r

  2. Crear un proyecto de Firebase con el mismo correo que la cuenta de GitHub.

  3. Configura Firestore y tu proyecto de Firebase para IoT.

  4. Edita el código, utilizando las diapositivas que siguen después de esta y pruébalo. Se prueba directamente sobre el sitio web.
