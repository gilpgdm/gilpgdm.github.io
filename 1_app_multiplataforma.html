Una App Nativa es una app desarrollada específicamente para una plataforma de cómputo.

Ejemplos de Plataformas

  • Web
  • Android
  • Windows
  • iOS (iPad y iPhone)
  • MacOS

Una App Multiplataforma es una app diseñada para ejecutarse en diferentes plataformas.

Tecnologías Multiplataforma

  • Progressive Web Application (PWA)
  • Cordova
  • Native Script
  • Ionic

App Nativa App Multiplataforma
1 código por plataforma. 1 código único para todas las plataformas. (Posiblemente en distintos lenguajes.)

  • Permiten que la experiencia de usuario se integre mejor al uso de la plataforma.

  • Mejor rendimiento.

  • Sacar mejor provecho a las características de la plataforma.

  • 1 solo código.

  • Menor tiempo de desarrollo.

  • Desarrollo más barato.
