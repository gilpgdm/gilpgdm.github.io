Instrucciones

  1. Crea los íconos del proyecto con GIMP.

  2. Crea la base de un sitio web usando una cuenta de Google, GitHub y opcionalmente Visual Sudio Code.

    Las instrucciones que se mencionan de git son:

    git config --global user.name tu_usuario_en_GitHub

    git config --global user.email tu_email_en_GitHub

    Para actualizar tu página cuando no han pasado 10 minutos del último despliegue, teclear

    Shift+Ctrl+r

  3. Edita el código, utilizando las diapositivas que siguen después de esta y pruébalo.

  4. Prueba, despliega, instala y usa tu PWA en Windows.

  5. Instala y usa tu PWA en Android.

  6. Instala y usa tu PWA en tu iPhone o en tu iPad.
