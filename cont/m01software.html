CP210x USB to UART Bridge VCP Drivers

  • Lo puedes descargar de https://www.silabs.com/developers/usb-to-uart-bridge-vcp-drivers.

  • Selecciona la pestaña DOWNLOADS y descarga el archivo CP210x Universal Windows Driver.

  • Desde una cuenta de administrador, descompácta el archivo y ejecuta el instalador que corresponda a tu computadora, que puede ser de 32 o 64 bits.

  • Video de Instalación.

Arduino IDE
Arduino core for ESP8266 WiFi chip
Arduino core for ESP32 WiFi chip
ArduinoWebSockets

  • Puedes descargar Arduino IDE desde https://www.arduino.cc/.

  • Selecciona la pestaña SOFTWARE y busca el DOWNLOAD para tu sistema operativo.

  • Una vez instalado Arduino IDE, entra a esta aplicación, selecciona el menú Archivo > Preferencias. Se abre un cuadro de diálogo y en el campo Gestor de URLs Adicionales de Tarjetas introduce el valor

    https://raw.githubusercontent.com/espressif/arduino-esp32/gh-pages/package_esp32_index.json,https://arduino.esp8266.com/stable/package_esp8266com_index.json
