• Los mensajes de error de de un componente, se muestran en otro componente conocido como texto de ayuda.

  • En algunos de los custom elements mostrados, se indica el texto de ayuda usando un elemento small con el atributo slot="supporting".

  • La forma usa el atributo novalidate.

  • En este ejemplo, el campo select-menu con el id inGenero usa el evento input. para invocar la función copiaMensajes. y copiar los mensajes de error del campo de captura al texto de ayuda.
