• Se usa una etiqueta flotante que se muestra grande cuando el contenido de campo es una cadena vacía y se muestra pequeña cuando el elemento se está capturando o cuando su contenido no está vacío.

  • Se el elemento personalizado md-filled-text-field, definimos con la clase MdFilledTextField.

  • Las etiquetas deben ser label o span.

  • Los campos de captura deben llevar al atributo slot="target".

  • Los textos de ayuda deben llevar al atributo slot="supporting" y normalmente son de tipo small..

  • Usa class="float" cuando quieras que la etiqueta esté arriba todo el tiempo.
