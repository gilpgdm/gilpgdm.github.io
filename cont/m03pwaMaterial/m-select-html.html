• Este control se adapta mejor a Material Designe que el select de HTML.

  • Usa el elemento personalizado md-select-menu, definido con la clase MdSelectMenu.

  • Coloca el elemento md-select-menu con slot="input-text", dentro de un md-filled-text-field. Usa un span cono etiqueta.

  • Para las opciones usa el elemento personalizado md-options-menu, definido con la clase MdOptionMenu. Dentro de él coloca elementos tipo label que en su interior llevan input type="radio" y el texto de la opción.

  • Para asignar un valor inicial, usa el atributo value del md-filled-text-field.

  • Para que forzosamente se deba seleccionar una opción, agrega el atributo booleano required al md-select-menu.
