• Se usa para seleccionar entre 2 o más vistas.

  • Se usa en:

    • Móvil

    • Tablet

    • Escritorio

  • Se crea el elemento personalizado mi-nav, que puede usarse en todas las vistas de la app.

  • En el método connectedCallback, añade la instrucción this.classList.add("navTab"), para indicar que usa pestañas.

  • Coloca mi-nav, usando el atributo slot="adicional". dentro de una barra de aplicación.

  • Avanzar a la parte del código dónde se usa.

Salida

Ábrelo en otra pestaña.

     }
    }
   }

      () => this._avanza())
    }
    this._scrolling = true
   }

    if (this._posY === 0) {
     this.classList.
      remove(ELEVATION_CLASS)
    } else {
     this.classList.
      add(ELEVATION_CLASS)
    }
    this._scrolling = false
   }
  }
