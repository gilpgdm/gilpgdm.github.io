El Elemento video

Se usa para desplegar videos o películas.

Aunque este elemento puede llevar contenido, por ejemplo texto, este no se muestra al usuario; es para desplegarse en navegadores que no soportan video, para poder probar con plugins o indicar como acceder al contenido.

Para que tu video se despliegue en el navegador Safari para las computadoras Apple (iPad, iPhone, Mac), tu servidor web debe soportar byte-range request. No todos los servidores soportan este tipo de solicitudes.

Una opción es usar el servicio storage de Firebase, que es gratis para archivos multimedia que juntos ocupen hasta GB y con poco tráfico. La URL que te proporciona el servicio es muy larga. La puedes acortar usando un servicio como Bitly.

Otra opción es usar Giphy que en formato de video es más rápido y eficiente que el formato GIF. Al seleccionar una imagen, haz clic en Media y copia la URL del formato MP4.

Atributos adicionales

poster

Proporciona la URL de una imagen que se muestra mientras no se haya cargado ningún byte del video.
