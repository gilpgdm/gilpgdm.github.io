14. one-line.html - Listas con elementos one-line

  • Las listas tienen los siguientes elementos:

    • Un encabezado que muestra texto en negritas.

    • Opcionalmente un texto secundario que muestra mas pequeño y de ancho normal.

    • Opcionalmente muestran una imagen a la izquierda, que puede ser:

      • Un pequeño ícono cuadrado.

      • Una imagen cuadrada.

      • Una imagen rectangular.

      • El avatar de un usuario. (Se muestra dentro de un círculo.)

  • Las listas deben usar un elemento ul con la class list.

  • Los elementos de la lista se colocan en elementos li.

  • Si el elemento es un enlace, se coloca un elemento a dentro del li.

  • Si el elemento solo lleva un renglón, se coloca la class one-line dentro del elemento li. o dentro del elemento a, si es un enlace.

  • Si el elemento es de 2 renglones, se coloca la class two-line dentro del elemento li. o dentro del elemento a, si es un enlace.

  • Si el elemento es de 3 renglones, se coloca la class three-line dentro del elemento li. o dentro del elemento a, si es un enlace.

  • Todos los elementos llevan un span con la class headline. para el título, así como un elemento span con la class supporting, para el texto secundario.

  • La imágenes se pueden agregar como un material symbol o como un elemento img. El elemento debe añadir una clase para indicar el tipo de imagen que lleva, que puede ser:

    • icon.

    • image.

    • video.

    • avatar.
