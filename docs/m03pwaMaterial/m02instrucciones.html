3. Hazlo funcionar

  • Descarga el ejemplo.

  • Revisa el proyecto en Replit con la URL https://replit.com/@GilbertoPachec2/pwamd?v=1. Hazle fork al proyecto y córrelo.

  • Prueba e instala el ejemplo con la URL https://gilpgpwamd.github.io/

  • Crea los íconos del proyecto con https://www.photopea.com/. Este paso se hizo com Microsoft Edge (versión Chromium), pero no funcionó con Google Chrome.

  • Crea los íconos enmascarables con https://maskable.app/ a partir del archivo «icono2048.png».

  • Usa o crea una cuenta de Google.

  • Crea una cuenta de Replit usando la cuenta de Google.

  • Crea un proyecto HTML, CSS, JS en Replit y edita o sube los archivos de este proyecto.

  • Entra al sitio https://m3.material.io/theme-builder#/custom, selecciona los colores de tu aplicación, haz clic en Export y selecciona Web (CSS) para descargar el zip que contiene los estilos que generan los colores. Descompacta el zip y copia el archivo tokens.css de la carpeta descompactada, para sobreescribir el archivo del mismo nombre que está en la carpeta css del proyecto.

  • Depura el proyecto.

    Cada vez que publiques los cambios en los archivos, debes modificar el valor de VERSION en el archivo «sw.js».

    Recuerda usar la combinación de teclas Ctrl+Mayúsculas+r para forzar que se actualice el navegador en caso de que no se vean los cambios.

  • Crea la cover page o página de spotlight del proyecto.

  • Instala y usa tu PWA en Windows. Aunque en este video se recomienda usar Edge, al momento de actualizar el contenido, la opción más recomendada es Chrome para que te muestre las descripciones y las capturas de pantalla.

  • Instala y usa tu PWA en Android. Al momento de actualizar las notas, tal vez no te aparezca el botón para instalar y tengas que seleccionar la acción de instalar que aparece en el menú de extensión de Chrome.

  • Instala y usa tu PWA en iOS (iPhone y iPad).

  • El proyecto incluye archivos de otras fuentes que se pueden descargar de la de su sitio original; para ello sigue estas instrucciones:

    • El font Roboto se descarga usando https://gwfh.mranftl.com/fonts. Usa la opción Legacy Support. Ahí mismo dice como crear el archivo de estilos.

    • Los íconos de Material Design se descargan del repositorio https://github.com/google/material-design-icons/tree/master/variablefont. Descarga los archivos MaterialSymbolsOutlined[FILL,GRAD,opsz,wght].codepoints, MaterialSymbolsOutlined[FILL,GRAD,opsz,wght].ttf y MaterialSymbolsOutlined[FILL,GRAD,opsz,wght].woff2.

    • En https://developers.google.com/fonts/docs/material_symbols. te indica como crear el archivo de estilos.

    • Descarga los tokens para Material Design de https://github.com/material-foundation/material-tokens/tree/main/css. Algunos archivos se modificaron para soportar iOS.

    • Otros archivos de tokens también se toman de la descarga de https://m3.material.io/theme-builder#/custom. Algunos archivos se modificaron para soportar iOS.

    • Descarga el archivo custom-elements.js de https://github.com/ungap/custom-elements. que sirve para usar buil in custom elements en iOS.
