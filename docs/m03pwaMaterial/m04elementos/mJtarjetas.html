11. tarjetas.html - Tarjetas

  • Las listas de tarjetas deben usar la class cards.

  • Pueden llevar multimedia debe colocarse dentro de un elemento de tipo figure.

  • El texto de encabezado se marca con la class headline.

  • El texto de soporte se marca con la class supporting.
